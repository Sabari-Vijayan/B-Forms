# Overview: Prompt-to-Form

Prompt-to-Form is a Google Forms alternative that leverages AI to simplify form creation and handle automatic multilingual support.

## 🌟 Key Features

### 1. AI-Powered Generation
Users can describe the form they want in natural language (e.g., "Create a customer feedback form for a taco shop"). The system use Google Gemini 1.5 Flash to generate a structured form with appropriate field types (short text, rating, single choice, etc.).

### 2. Multilingual Support
The platform is designed to be language-transparent:
* **Creators** design forms in their preferred language.
* **Respondents** can view and answer the form in any supported language.
* **Automatic Translation**: Fields are pre-translated on publish, and responses are translated back to the creator's language asynchronously.

### 3. Drag-and-Drop Editor
A full-featured editor allows creators to refine AI-generated forms, reorder fields, and add new ones manually.

### 4. Real-time Submissions
Creators can view responses as they come in, with translation status indicators (pending, done, failed).

### 5. Sharing and Analytics
Forms get a unique shareable URL and QR code. Creators have access to a dashboard with response statistics and language breakdowns.
